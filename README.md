## 📖 Project Overview

* This project analyzes ride booking data from Ola using SQL and Power BI to uncover trends in bookings, revenue, cancellations, customer behavior, and driver performance.
* The goal of this project is to transform raw ride-booking data into actionable business insights through data cleaning, SQL analysis, and interactive Power BI dashboards.
* The dashboard enables stakeholders to monitor operational performance, identify cancellation patterns, evaluate vehicle-type performance, and understand customer satisfaction metrics.

## 🎯 Business Problem

Ride-hailing companies generate large volumes of booking data every day. However, without proper analysis, it becomes difficult to:

- Monitor booking trends
- Track revenue performance
- Understand cancellation behavior
- Evaluate vehicle category demand
- Measure customer and driver satisfaction

This project addresses these challenges by providing an interactive analytics dashboard that supports data-driven decision-making.

## 🔍 Key Insights

- Successfully completed rides account for approximately 67% of all bookings.
- Overall cancellation rate is approximately 27%.
- Revenue is distributed across multiple payment methods.
- Different vehicle categories contribute significantly to total booking value.
- Customer and driver ratings remain consistently around 4.0, indicating strong service quality.
- Cancellation reasons provide valuable operational improvement opportunities.
